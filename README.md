# Trivillig – Manual Assets

Skærmbilleder brugt i brugermanualen for **Trivillig Mobil** (den frivilliges app).

Billederne ligger i `screenshots/` og indlejres i Notion-manualen via deres `raw`-URL'er.

> Personhenførbare data (ansigter, adresse) er sløret af hensyn til privatliv.
> Indhold stammer fra en testkonto i udviklingsmiljøet.
